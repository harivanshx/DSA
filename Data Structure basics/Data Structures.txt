Interview Pattern #1 â€” Reverse an array
Classic starter question.
Logic

Swap front â†” back.

def reverse(arr):
    left = 0
    right = len(arr) - 1

    while left < right:
        arr[left], arr[right] = arr[right], arr[left]
        left += 1
        right -= 1

    return arr


Input:

[1,2,3,4]


Output:

[4,3,2,1]
